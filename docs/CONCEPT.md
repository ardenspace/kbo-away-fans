> (Arden의 초기 브레인스토밍 메모를 정리한 것)

## 한 줄 컨셉

**팬도 원정을 간다!** — 원정 간 KBO 팬들을 위한, 원정 여행을 즐길 수 있게 해주는 앱.

## 톤 & 무드

- **게임 컨셉**: 애니메이션 효과 많이 (인터랙티브하게)
- **디자인 컨셉**: 야구

## 기능 (초안)

### 1. 맛집 추천
- **수작업/크롤링**: 유튜브·인스타·커뮤니티에서 "선수 맛집" 키워드로 나오는 걸 LLM으로 요약·정리, 출처 링크 붙여 DB 적재
- **유저 제보**: 출처 URL 필수 → 어드민 승인 후 노출
- **반자동 크롤링**: 네이버 카페/갤러리/유튜브 자막 등 주기적으로 긁어 LLM으로 "선수 + 장소" 멘션 추출 → 후보 리스트를 어드민 대시보드에 띄우는 파이프라인
  - → 어드민 대시보드 필요 (pslog 프로젝트 활용 or 텔레그램으로 후보 전달)

### 2. 구장별 가이드
- 주차 팁, 좌석 뷰 사진, 입장 동선, 근처 편의점/약국 위치

### 3. 경기 일정 연동
- 응원 팀 알면 → 해당 팀 경기 일정 연동 → 원정 지역 추천 자동 노출

### 4. 우천 대비 플랜B
- 방탈출, 야경, 소품샵, 독립서점 등 구장 반경 내 실내 놀거리를 카테고리로 묶어 제공

### 5. 원정 체크리스트
- 방문 구장·맛집 스탬프 → 10구장 완주 뱃지

### 6. 팬 후기 / 사진
- 장소별 미니 리뷰, 커뮤니티 사진 리뷰

## 스택 (초안)

- **앱**: Flutter — 자체 렌더링 엔진으로 복잡한 애니메이션 자유도·성능 안정
- **DB**: Supabase
- **크롤링 + LLM 파이프라인**: 맥미니에서 구동

### 욕심나는 애니메이션
- 구장 스탬프 랠리 (방문 구장에 도장 찍히는 모션)
- 스크래치 카드 UI ("오늘 뭐 하지?" 긁으면 랜덤 추천)
- 구장별 테마 컬러 전환 (대전=주황, NC=민트 … 팀 컬러 부드럽게 전환)
- 지도 위 경로 애니메이션 (원정 기록을 지도 선으로 이어 이동 경로 그리기)
- 맛집 카드 스와이프 (원정 맛집 리스트 만들기)
- 날씨 연동 파티클 (비 오면 빗방울 파티클 → 자연스럽게 플랜B 화면 유도)

## 열린 결정 사항 (기획에서 확정 필요)

1. **LLM 선택**: 로컬 LLM(맥미니/llama.cpp) vs 헤르메스 에이전트(with Codex) — 비용/품질/운영 트레이드오프
2. **어드민 대시보드**: pslog 재사용 vs 텔레그램 봇 vs 별도 웹
3. **MVP 범위**: 6개 기능 중 첫 출시에 뭘 넣을지 (스탬프+맛집+일정연동이 코어 후보?)
4. **데이터 시딩**: 초기 맛집/구장 데이터를 어떻게 채울지 (수작업 vs 크롤링 우선순위)
5. **백엔드 경계**: Supabase 직접 호출 vs 별도 API 레이어(크롤링·LLM 파이프라인용)
