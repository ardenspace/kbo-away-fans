# kbo-away-fans

> 🚧 초기 셋업 단계. 컨셉/스택은 pslog-workflow 플러그인의 기획 단계에서 확정 예정.

## 개발 방식

이 프로젝트는 [pslog-workflow](https://github.com/ardenspace/pslog) 플러그인을 사용해
**기획 → 실행 → 리뷰** 워크플로로 진행한다.

- `pslog-planning` — 아이디어 → 실행계획 → PLAN.md 분해
- `pslog-workflow` — PLAN task → 무게 게이트 → 코드
- `pslog-refactor` — 코드 진단 → 동작 보존 리팩토링
